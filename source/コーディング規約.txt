変数


int someThing;
int someThings[10];
std::list<int> numberList;
std::list<int> numbers;

頭文字は小文字で、そのあとのワードの頭文字は大文字でお願いします。
例外として「ID」「UFO」などの固有略称は位置問わず全部大文字でもいい。

配列やリストなどは複数系を使ってください。
あるいは「なんとかリスト」「なんとかArray(配列)」で、要は単体じゃないって分かればいい。

g_、m_、p
グローバル、メンバ、ポインタ、その他型を示すシンボルなどは、
名前被らない限り付けなくてよい。

for(int i = 0; i < 10; i++) {

}
for文のコントロール変数は「i」を優先的に使ってください。（indexって意味で）
多重ループや、コントロール変数が添字として使われていない時はその限りじゃない。

float型の値は必ず小数点とfをつけてください。
0 → 0.0f

========================================================================================

関数


void DoSomeThings(int x);

void DoSomeThings(int x)
{
	// ...
}

void DoSomeThings(float x)
{
	// ...
}

頭文字は大文字でお願いします。
多少長くてもいいから、名前はなるべくはしょらないでください。
{}は改行してください。
(if文, for文などは改行しなくていい)

========================================================================================

ヘッダー


#ifndef _GAME_OBJECT_H
#define _GAME_OBJECT_H

#include "..."

// ...

#endif

インクルードガードは#ifndef式を使ってください。
define文字列はファイル名を全部大文字にして、
ワードの間、ドット、先頭に「_(アンダースコア)」を入れてください。
他のヘッダーのインクルードは　必ず　インクルードガード内に書いてください。

あと、クラスのインライン関数は禁止。
空の{}でも関数の定義（中身）は必ずcppファイルに書いてください。
逆に宣言と定義は同じcpp内に完結するのは可。

========================================================================================

定数


#define SOME_CONSTANT (10.0f)
constexpr const char* SOME_CONSTANT_EXPR = "word";

全部大文字にアンダースコア
基本的にconstexprの方を推奨だが、状況によってマクロ使っても全然アリ。

その他


文の途中で改行するならコンマなどの記号を行末に置いてから改行してください。
記号のあとはスペース一つ挟んでください。

↓ダメ
DoSomeThings(
	int x,int y
	,int z,int w
);

↓ヨシ
DoSomeThings(
	int x, int y,
	int z, int w
);

↓ダメ
int x=(1+1)*2;

↓ヨシ
int x = (1 + 1) * 2;

staticじゃないグローバル変数は禁止！
